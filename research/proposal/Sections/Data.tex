\subsection{Data}
The dataset must comprehend be as detailed as possible. The ideal $\vec{\Delta}$ would be 15 minutes, but 1 hour would also be functional. It remains to be seen if a larger $\vec{\Delta}$ of 1 hour with statistical summaries
is benefitial, and it might be worth testing.

The dataset should include the price and energy production/consumption for the the netherlands. Weather data for the Netherlands would be ideal. We could cluster different sensor points, and use location as a feature,
however we should start simpler, by using a national average. Temporal features should also be included, such as weekend, or holiday flag. All features should be interpolated on timestamp.

This dataset would be good enough for modelling.
